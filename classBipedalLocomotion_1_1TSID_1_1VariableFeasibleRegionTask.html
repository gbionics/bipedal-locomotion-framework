Detailed Description

VariableFeasibleRegionTask is a concrete implementation of the Task.

Please use this element if you want to create a inequality linear constraint on a subset of the optimization variable. The task represents the following equation:

\[ l \ne Cx \ne u \]

where \(C\) is a generic transformation matrix (m x n), \(l\) is a vector of lower bounds (m x 1), \(u\) is a vector of upper bounds (m x 1), and \(x\) are the elements of the variable you want to consider (n x 1). \(m\) is the number of constraints and \(n\) is the number of variables.

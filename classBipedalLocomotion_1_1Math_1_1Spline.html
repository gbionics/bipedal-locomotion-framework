Detailed Description

template<typename T>
class BipedalLocomotion::Math::Spline< T >

Spline implement a basic spline.

Note
The spline is defined as a set of polynomials. The coefficients of the polynomials are computed using the boundary conditions. The boundary conditions are the initial and final velocity and acceleration.
Template Parameters
Ttype of the spline. T must be an Eigen type (i.e. Eigen::Vector3d).

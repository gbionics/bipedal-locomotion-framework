Detailed Description

Struct defining a contact phase.

Each phase is characterized by a set of contacts which remain active for the entirety of the phase.

Note
Mathematically speaking the interval of the phase is defined as following

\[ I = [t_b \; t_e) \]

where \(t_b\) is the ContactPhase::beginTime and \(t_e\) is the ContactPhase::endTime. The end time is not included in the phase.

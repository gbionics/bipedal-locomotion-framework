Detailed Description

UkfModel is a base class to define the process and measurement model of the UKF.

It includes all the methods shared between the two concrete classes, as the update of the robot dynamics and the robot state.

std::map<std::pair<std::string, std::string>, std::string> BipedalLocomotion::Estimators::RobotDynamicsEstimator::UkfModel::m_variableNameToUkfState
protected

Map used to retrieve the name of the variable passed as state and the ukf name.

The key is a pair of <name of the variable in the state vector, sensor type (ft, accelerometer, gyroscope)> and the value is the dynamics group name that is the name used in the configuration file to define the group associated to the dynamics.

Detailed Description

Computes the unique stabilizing solution S to the continuous-time algebraic equation.

\[ S A + A' S - S B R^{-1} B' S + Q = 0 \]

Note
This class implements the algorithm presented in the paper: Solving the algebraic Riccati equation with the matrix sign function https://www.sciencedirect.com/science/article/pii/0024379587902229

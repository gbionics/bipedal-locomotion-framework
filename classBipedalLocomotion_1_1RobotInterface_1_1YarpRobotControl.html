Parameters
desiredJointValuesdesired joint values.
controlModesvector containing the control mode for each joint.
Returns
True/False in case of success/failure.
Note
In case of position control the values has to be expressed in rad, in case of velocity control in rad/s. If the robot is controlled in torques, the desired joint values are expressed in Nm. If the robot is controlled in PWM, the desired joint values are between -100 and 100.
Warning
At the current stage only revolute joints are supported.
